# faq-chatbot

FAQ Chatbot is a machine learning based chatbot trained on FAQ [https://www.kaggle.com/datasets/abbbhishekkk/faq-datasets-for-chatbot-training?select=HDFC_Faq.txt) of Hdfc Bank.

- Libraries used:
  - Scikit-Learn
  - NLTK
  - Re
  - Numpy
  - Pandas
  - NLPAug
